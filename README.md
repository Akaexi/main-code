LANGKAH - LANGKAH WAIRING DAN INSTALASI!!
1. WAIRING ANTARA ESP32 DENGAN LAYAR ST7789
     GND ke  GND
     VCC  ke  3,3v/5v
     SCL   ke  D18
     SDA  ke  D23
     DC    ke  D2
     RST   ke  D4
     CS      -1 (tidak digunakan)
2. UPLOAD CODE play_lyric_lagu_ST7789.ino ke esp32 
3. BUKA CODE LAGU .py DENGAN VS CODE DAN RUN 
4. JANGAN LUPA SESUAIKAN PORT ESP DENGAN KODE PYTHON
